## 🚀 Getting Started

### 🔧 Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher) & npm
- [Python 3.8+](https://www.python.org/)
- [MongoDB](https://www.mongodb.com/try/download/community) (v4.4 or higher)
- [Git](https://git-scm.com/)

### 🔑 Environment Setup

1. **Copy the environment template:**
   ```bash
   copy env.example .env
   ```

2. **Configure your environment variables:**
   ```bash
   # Edit .env file and configure:
   GEMINI_API_KEY=your_actual_gemini_api_key_here
   MONGODB_URI=mongodb://localhost:27017/serene
   JWT_SECRET=your-super-secret-jwt-key-change-this-in-production
   ```

   **Notes:**
   - The Gemini API key is optional. If not provided, the system will use the Python backend as a fallback.
   - Make sure MongoDB is running on your system.
   - Change the JWT_SECRET in production for security.

### 🧪 Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd Serene-Complete
   ```

2. **Install Node.js dependencies:**
   ```bash
   npm install
   ```

3. **Install Python dependencies:**
   ```bash
   pip install -r requirements.txt
   pip install fastapi uvicorn
   ```

### 🚀 Running the Application

#### **Option 1: Use Startup Scripts (Recommended)**

**Windows:**
```bash
start.bat
```

**PowerShell:**
```bash
.\start.ps1
```

#### **Option 2: Manual Startup**

**Terminal 1 - Start Python FastAPI backend:**
```bash
npm run python
```

**Terminal 2 - Start Node.js server:**
```bash
npm run server
```

### 🌐 Access Points

- **Main Application:** http://localhost:3000
- **Python API:** http://localhost:8000
- **API Documentation:** http://localhost:8000/docs
- **Health Check:** http://localhost:3000/health

## 🔧 API Endpoints

### Authentication
- `POST /api/login` - User login
- `POST /api/signup` - User registration
- `GET /api/profile` - Get user profile (protected)

### Chat
- `POST /api/chat` - Send message to AI companion (protected)
- `GET /api/chat/history` - Get chat history (protected)

## 🆘 Support

If you encounter any issues:

1. **Check that all services are running:**
   - Node.js server (port 3000)
   - Python FastAPI server (port 8000)
   - MongoDB database (port 27017)

2. **Verify your environment variables:**
   - Check `.env` file exists and is properly configured
   - Ensure MongoDB connection string is correct
   - Verify JWT secret is set

3. **Check the console for error messages:**
   - Look for database connection errors
   - Check for authentication issues
   - Verify API endpoint responses

4. **Ensure all dependencies are installed:**
   - Run `npm install` for Node.js dependencies
   - Run `pip install -r requirements.txt` for Python dependencies
   - Make sure MongoDB is installed and running

5. **Database troubleshooting:**
   - Ensure MongoDB service is running: `mongod`
   - Check database connection: `mongo` or `mongosh`
   - Verify database permissions and access
